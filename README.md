# Whiteboard App

A browser-based whiteboard application for drawing, sketching, and visual note-taking using an interactive canvas.

ðŸ”— **Live Demo:** https://whiteboard-app-lovat.vercel.app/

---

## Features

- Freehand drawing (brush)
- Shape tools: line, rectangle, circle, arrow
- Text tool
- Undo / Redo
- Eraser tool
- Export canvas as PNG

---

## Tech Stack

- React
- Context API + useReducer
- HTML Canvas
- Rough.js
- perfect-freehand
- Tailwind CSS + CSS Modules
